About the Project

This project provides a comprehensive analysis of donation and consumer data. Through visualizations and forecasting models, we aim to highlight key trends, seasonal patterns, and future projections.

It leverages modern technologies, including Flask, Python, XGBoost, and Matplotlib, to deliver meaningful insights and user-friendly interactive features.

Key Features:

  • Time-series visualizations for monthly trends.
  • Exploratory Data Analysis (EDA) to uncover hidden patterns.
  • Interactive forecasting using advanced models like XGBoost.
  • Dynamic user input for custom forecasts.

Why This Matters:

Understanding donation and consumption trends is crucial for efficient resource allocation and demand forecasting. This tool empowers users with actionable insights to improve decision-making.
